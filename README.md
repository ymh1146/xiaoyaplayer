# 小雅 Alist 播放器

一个专为小雅 Alist 设计的桌面播放器，提供流畅的流媒体播放体验。（只能看到视频媒体，其余文件已自动忽略）

## 核心功能

- **WebDAV 直连**：无需下载，直接播放
- **智能排序**：自动识别剧集命名（SxxExx、第xx集等）
- **自动连播**：播放完自动下一集
- **片头片尾跳过**：单击设置跳过时间，长按重置
- **播放历史**：自动恢复上次播放位置
- **现代化 UI**：暗色主题，自动隐藏控制栏
- **全屏模式**：鼠标移动唤醒控制栏

## 快捷键

| 功能 | 快捷键 |
| --- | --- |
| 全屏/退出 | `Enter` |
| 播放/暂停 | `Space` / 双击视频 |
| 音量 ±5 | `↑` / `↓` |
| 快进/快退 15秒 | `→` / `←` |
| 上一集/下一集 | `Ctrl+Z` / `Ctrl+X` |

## 安装使用

1. **安装 VLC 播放器**（必需）
    ```bash
    https://www.videolan.org/vlc/
    ```
    说明：不使用webplayer的原因是一些超清剧集会自动降低分辨率播放，具体原因不明，因此选用VLC作为播放器

2. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

3. **运行程序**
   ```bash
   python main.py
   ```

4. **连接服务器**：在界面左上角输入 WebDAV 地址并点击"连接"
